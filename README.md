# CV Maker

A simple CV/Resume generator built with React. It allows users to fill in their personal information, education history and work experience, and preview the final CV in real time.

Live demo on [Cloudflare Pages](https://cv-maker-2oc.pages.dev/)

## Technologies

React

JavaScript

CSS

Vite

## Getting Started

```sh
git clone git@github.com:JosiasPires/resume.git
cd resume
npm install
npm run dev
```
